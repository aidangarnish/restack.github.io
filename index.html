RESTACK Developer Guide

RESTACK is the universal interface for connected devices

Welcome

HTTP REST API: (base URI: https://alpha.restack.io) Our API endpoints require authentication credentials to be passed in the HTTP headers as restack_auth_uuid and restack_auth_token.

  1. Go to https://dashboard.restack.io/register and create an account
  2. Go to Devices -> New device and add a new device. You will receive an UUID for your device and a token.
  3. Use this credentials to send data.

Endpoints

Status (method: GET)

Returns the status of RESTACK API service

curl -X GET https://alpha.restack.io/status

response:

{
  "restack": "online"
}

IP (method: GET)

Returns the public IP address of the request

curl -X GET https://alpha.restack.io/ipaddress

response:

{
  "ipAddress": "127.0.0.1"
}

DATA (method: POST)

Stores device/sensor data for a specific UUID. You can pass any key/value pairs.

curl -X POST -d "luminosity=102&temperature=-5" https://alpha.restack.io/data/520ca5d1-a1e9-11e4-903e-c1e2ccd2ffa5 --header "restack_auth_uuid: {uuid}" --header "restack_auth_token: {token}"

response:

{
  "timestamp": "2015-01-22T05:30:02.375Z",
  "uuid": "520ca5d1-a1e9-11e4-903e-c1e2ccd2ffa5",
  "temperature": "-5",
  "luminosity": "102",
  "ipAddress": "188.30.124.60",
  "eventCode": 700,
  "_id": "54c08ada3d61a74760db80c3"
}
